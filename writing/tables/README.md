# Tables

Standard for every table in a paper. **Planned — not yet written.**

Will cover: `booktabs` rules (no vertical rules, `\toprule`/`\midrule`/`\bottomrule`),
Econometrica table style, **generate-don't-type** (table bodies emitted by a
committed script in the paper repo, never hand-typed), number formatting and
alignment, notes/source lines, and `\label{tab:<name>}` conventions.

Until this chapter lands, follow the figures chapter in spirit: reproducible from
a committed script, document serif fonts, self-contained caption and notes.
