Runner dashboard

Monitoring & control UI for the Standards distributed runner. It reads a project repo's task queue, solutions, and figure data directly from GitHub. Currently pointed at mhpbreugem/MIWN (queue todo/TASK_QUEUE.json, pool solutions/pool/, figures solutions/figures_data.json).

Status

Queue progress, per-task state, claimed-by worker, results, and figures.

Workers

Active workers and which tasks they hold.

Mobile / control

Compact view; dispatch the solve workflow and manage runs.

To point this at a different project, edit the OWNER/REPO/BRANCH constants at the top of each page. Part of standards/runner/.
